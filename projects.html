Academic Projects

IMMI APP - Android App

Part of Android Mobile Application Development class in Spring 2015. I worked on a team of three on conceptualizing, designing and developing a mobile application on the Android SDK.
Immi is a location based social media app that allows users to view and create events, mainly locally, within a category and a customizable radius and uses a rating system to view best events. Source Code

Technologies used: Android Studio, Java, and external services such as Google Maps and Facebook API.

MARIO MAZE - 2D Windows Game

MARIO MAZE is a 2D Interactive Game Design developed during CSE 3902 project calss in August - December 2014. I developed level 1-1 of original 2D Mario Bros game using .NET framework, C#, XNA and Visual Studio. During the development of the game my team members and I followed the Agile Software Development methodology over the period of 13 weeks. In addition to increasing our knowledge on Design Patterns and Code Quality. Source Code

Technologies used: .NET framework, C#, XNA, XML and Visual Studio.

Hackathons

Bonfire App - Event Sharing App @ Giveback Hack

Bonfire is an app that increases millennial volunteerism by letting users know what events their friends are going to. The app is build on the idea that people are more likely to volunteer when they see that their firends are volunteering. My team and I build an Android app, an iOS app and a webapp for our project within a weekend at Giveback Hack. The site is also on heroku.

Technologies used: Android Studio, Java, Google Maps.

MedAssist - Assistant for Medicine Adherence @ Fuse - Cardinal Health

I participated in a 24 hour long hackathon at Fuse by Cardinal Health to hack on healthcare solutions. My idea was to make a webapp that the elderly can use to track their medication. My team used a sound recognition framework to help users navigate through the page and find their medications. Here's the source code for the hack, and a link to the demo site. BTW, try to visit that site and allow microphone and say "show my evening pills" and see what happens.

Technologies used: HTML, CSS, Javascript, and Bootstrap
